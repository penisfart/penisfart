Exploring Computer Tech Terms

Define the following including pictures below if applicable. If the item doesn’t exist list as “non existent”:

  1. Authentication

Authentication verifies the identity of a person or device. An example would be a website asking for a username and password:

  1. Intel's Thunderbolt

Allows the connection of peripherals to a computer.

  1. Apple's Lightning Connector

A cable that charges and gives power to apple devices.

  1. Boot sector

A boot sector is a physical sector, or section, on a hard drive that includes information about how to start the boot process[1]


  1. mSATA port

an interface connector that enables more effective Serial ATA (SATA)[2]

  1. M.2 Port

M.2 is a slot that can interface with SATA 3.0, PCI Express 3.0, and USB 3.0.[3]

  1. U.2 / Mini SAS port

a computer interface standard for connecting SSDs to a computer.[4]

  1. RS-232 port

 a serial port is a serial communication interface through which information transfers in or out sequentially one bit at a time.[5]

  1. CMYK

Cyan, Magenta, Yellow, and Black, are the four basic colors used for printing color images.[6]

  1. NVidia Shield

digital media player produced by Nvidia

  1. ACPI

An ACPI (Advanced Configuration and Power) Interface specifies how a computer's basic input/output system, operating system, and peripheral devices communicate with each other about power usage[7]

  1. FIoS

Fios is a Verizon owned internet company that provides a 100% fiber-optic network[8]

  1. NuBus Slot0

a 32-bit parallel computer bus[9], which is a communication system that transfers data between components inside a computer[10]

  1. Vesa Local Bus Slot

 an older local bus architecture used in 1993 and 1994.[11]

  1. MicroChannel Slot

a bus standard for IBM PC compatible computers.[12]

  1. EISA Slot

a bus standard for IBM PC compatible computers.

  1. Firewall

a network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.[13]

  1. USB Type B

One of two standard USBs (Universal Serial Bus), roughly square in appearance, with a squarish protrusion on top.[14]

  1. GPU

Stands for graphics processing unit, allows for creation of images in a frame buffer intended for output to a display device.[15]

  1. Micro USB

a miniaturized version of the USB (Universal Serial Bus)[16]

  1. ATI CrossFire Bridge

a short cable that runs between two or more graphics cards[17]

  1. Hub

a basic networking device that connects multiple computers or other network devices together.[18]

  1. DB9 Serial Port

A type of serial port similar to the RS-232 port.

  1. Nvidia SLI Bridge Cable

a brand name for a multi-GPU technology developed by Nvidia for linking two or more video cards together to produce a single output.[19]

  1. RJ-45

 a type of connector commonly used for Ethernet networking[20]

  1. HDMI

High-Definition Multimedia Interface, proprietary audio/video interface for transmitting uncompressed video data[21]

  1. VGA

Video Graphics Array, similar to HDMI in that it transmits video data

  1. PS/2

a 6-pin mini-DIN connector used for connecting keyboards and mice to a PC compatible computer systems[22]

  1. ROM

Read-only memory

  1. RAM

random access memory, short-term data storage; it stores the information your computer is actively using so that it can be accessed quickly.[23]

  1. Cache

a hardware or software component that stores data so that future requests for that data can be served faster.[24]

  1. USB Flash Disk

 a data storage device that includes flash memory with an integrated USB interface.[25]

  1. CPU

central processing unit is the electronic circuitry within a computer that executes instructions that make up a computer program.[26][27]

  1. Motherboard

holds, and allows, communication between many of the crucial electronic components of a system, such as the central processing unit.[28]

  1. SATA Connectors

a computer bus interface that connects host bus adapters to mass storage devices such as hard disk drives, optical drives, and solid-state drives.[29]

  1. Computer Ports

External devices are connected to a computer using cables and ports.[30]

  1. PSU

        power supply unit, converts mains AC to low-voltage regulated DC power for the internal components of a computer. Modern personal computers universally use switched-mode power supplies.[31]

  1. DIMM

DIMM is a module that contains one or several random access memory (RAM) chips on a small circuit board with pins that connect it to the computer motherboard.[32]
